Client and server-side PDF Reporting

  • Client/server side: PDF printing in pure JavaScript (using the library jsPDF) and Python (using the library Pypdf)
  • Fully declarative: can be easily modified and used for relative layout (no need to specify the x and y coordinates)
  • Powerful layout engine: row, datagrid, column, cell, image, separator, html, barcode, hline, vgap elements
  • Many types of declaration: XML, JSON, Javascript or Python languages
  • Several output options: PDF, XML, HTML
  • Documentations: Reference Manual, Python DEMO, Javascript DEMO
  • Can be used independently: any (client or server-side) python and html5 (in-browser) programs
