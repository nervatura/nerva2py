XML template

  • report
    General report information.
    Attributes:
    - autor, creator, subject, title
    - left-margin, right-margin, top-margin
    - font-family: courier,times,helvetica - default times
    - font-style: if missing regular, or B(bold); I(italic); U(underline) or any combination
    - font-size: default 12
    - color, border-color, background-color
    - decode, encode: only PDF output, default value utf-8 and latin_1
  • header, details and footer
    Page header and footer: on every page
  • row
    Horizontal logical group. The last element width extends up to the right margin.
    Attributes:
    - height
    - hgap: default gap between these two elements
    - visible: table data source name - only if there is data
    • cell
      Attributes:
      - name: XML output node name (only details section, default "head" and except when name="label")
      - value: static text or databind value
      - width: if missing or 0, the automatic
      - border: 0(no border, default) and 1(frame) or some or all of the following characters: L(left); T(top); R(right); B(bottom)
      - align: L(left, default value); R(right); C(center); J(justification, only multiline)
      - multiline: if true, print text with line breaks (default false)
      - link: URL
      - font-family: default arial
      - font-style: if missing regular, or B(bold); I(italic); U(underline) or any combination
      - font-size: default 12
      - color, border-color, background-color
    • image
      Attributes:
      - file: path or URL of the image
      - width: if missing or 0, the automatic
      - link: URL
    • separator
      Vertical separator line
      Attributes:
      - gap: greater than 0 = double line
  • hline
    Horizontal line.
    Attributes:
    - width: if missing or 0, then extends up to the right margin
    - gap: greater than 0 = double line
  • vgap
    Vertical gap
    Attributes:
    - height
  • html
    See simple HTML rendering for reports introduction
  • datagrid
    Create a HTML table from python list (dictonary items)
    Attributes:
    - width
    - name: XML output node name (only details section, default "items")
    - databind: table data source name
    - border: 0(no border) or 1(frame, default)
    - color, border-color
    - font-size
    • header
      Opens each page
      Attributes:
      - background-color
      - merge: 1(all fields will be displayed in a single column) or 0(default)
    • footer
      Closes each page
      Attributes:
      - background-color
    • columns/column
      Attributes:
      - fieldname: dictonary key (special value: counter) - width: value in percent
      - label: column caption
      - align: L(left, default value); R(right); C(center); J(justify)
      - thousands: character, only number format
      - digit: number, only number format
      - footer: static text or databind value
